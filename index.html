My Frontend Web Development Journey

Documenting the topics and concepts I have learned in Module I: HTML & CSS Basics

1.1 Course Overview & Orientation

This lesson provides an introduction to the course, explaining the roadmap, objectives, and learning outcomes. It helps students understand what frontend development entails and how the modules are structured.

1.2 What is Frontend web development?

Frontend development is about creating the user interface of a website ‚Äî what users see and interact with. HTML defines structure, CSS defines style, and JavaScript defines behavior. This lesson explains how these technologies work together.

1.3 Building a Hobbies Webpage

In this topic, I learned to create my first simple webpage showcasing my hobbies. I practiced using headings, paragraphs, lists, and images, understanding how browsers render HTML content.

1.4 Setting up development environment - I

This lesson covers setting up tools to write and test code efficiently. Key learning: installing Visual Studio Code, organizing project folders, and introduction to GitHub for version control.

1.5 Setting up development environment - II

Continuation of environment setup with focus on enhancing productivity. Installing VS Code extensions like Live Server and Prettier, learning code formatting and shortcuts.

1.6 Introduction to Git & Github - I

This topic introduces Git, a version control system, and GitHub, an online repository hosting platform. Learned to initialize a Git repository, make commits, and understand version control.

1.7 Introduction to Git & Github - II

Focus on connecting local Git repositories to GitHub, creating a remote repo, pushing commits, and keeping code synchronized online.

1.8 Introduction to Git & Github - III

Advanced Git concepts for collaboration: branching, merging, and pull requests. These skills are crucial for team projects and professional workflows.

1.9 Let's play a game with Git commands

This lesson is a practical exercise to reinforce Git commands: forking, cloning, and raising pull requests to simulate real-world collaboration.

1.10 Introduction to HTML

HTML (HyperText Markup Language) is the backbone of web development. Learned basic structure of HTML document, common tags, and attributes. HTML provides the skeleton of a webpage.

1.11 HTML Block & Inline Elements

HTML elements are either block-level or inline. Block elements start on a new line and take full width (div, p, section), while inline elements stay within a line and take only the necessary width (span, a, strong). Understanding these helps structure content and layout.

1.13 Introduction to CSS

CSS (Cascading Style Sheets) controls how a webpage looks. Linked to HTML, it changes colors, fonts, backgrounds, and layout. CSS separates content from style for cleaner code.

1.14 Styling Elements using CSS selectors

CSS selectors target elements to apply styles. Examples: element selector (p {color: blue;}), class selector (.title {}), ID selector (#header {}). Selectors allow precise styling and customization.

1.16 HTML Tables

Tables structure data in rows and columns: table, tr, th, td. Attributes like colspan and rowspan merge cells. Tables are used for schedules, pricing charts, and data presentation.

1.17 HTML Forms

Forms collect user input: form wraps input elements, input types include text, email, password, checkbox, radio. Labels describe fields, textarea for multi-line input, button to submit. Forms are essential for login, feedback, and interaction.

1.18 CSS Box Model

The Box Model defines spacing for each element: content, padding, border, margin. Understanding this is essential for layout design and positioning elements properly.
